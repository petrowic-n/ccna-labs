# Lab 03: Basic Router Security

## Lab Info
- Level: Begginer
- Topic: Network Fundamentals
- Tool: Cisco Packet Tracer

## Goal
The goal of this lab is to practice basic security on router by adding them a password, encrypting it and decrypting it.

## Steps

1. Connect R1 and R2 by GigabitEthernet0/0 interfaces
2. Set hostnames according to network diagram
3. Set the enable password to each router to "cisco"
4. View the password in the running config, is it encrypted?
5. Enable password encryption on each router
6. View the password in the running config, is it encrypted?
7. Disable password encryption on each router
8. View the password in the running config, is it encrypted?

# Lab 01: Basic Network Connectivity

## Lab Info
- Level: Begginer
- Topic: Network Fundamentals
- Tool: Cisco Packet Tracer

## Objective
The goal of this lab is to build a simple network topology and establish communication between end devices.

By completing this lab I practiced:
- basic device setup in Cisco Packet Tracer
- Connecting network devices using correct cables
- Assigning IP addresses to end devices
- Verifying network connectivity

## Requirements
### 1. Build the Network

Add the following devices to your workspace
- 1 Router
- 1 Switch
- 2 PCs

### 2. Connect the Devices
- Connect each PC to the switch
- Connect the switch to the router
- Use appropriate cable types

### 3. Configure IP Addresses

Assign IP addresses to both PCs:
- Each device must be in the same network
- Use a valid subnet mask
- Define a default gateway

### 4. Verify Connectivity

- Test communication using ping command

## Success Criteria
- Both PCs can successfully communicate with each other
- No packet loss during testing
